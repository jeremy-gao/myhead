# MyHead
myhead是一个简易的ElasticSearch web可视化管理工具,可以用来观察集群状态，索引管理等.

# 功能介绍
---------------
 - 支持集群状态健康健康
 - 分片数监控
 - 索引最大前20(数量可调整)
 - 查看索引mappin，stats，
 - 删除，和批量删除索引


# 计划支持功能
---------------
 - 集群状态非Gree告警
 - 多方面绘图统计等.
